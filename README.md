## 🚀 How to Start (5 Steps)

**Step 1 — Read `HOW-TO-USE-CLAUDE-PROJECTS.md`**
Understand what a Claude Project is and how to configure it.
Free account works. Claude Pro recommended for uninterrupted sessions.

**Step 2 — Set up the MCQ Project once**
Read `MCQ-Project-Setup.md`. One project, all subjects, forever.

**Step 3 — Open your subject folder**
Go to `Section A/` or `Section B/`, pick a subject, open
`[Subject]-Project-Instructions.md`. Copy the prompt. Paste into Claude.

**Step 4 — Upload the Revision PDF**
Each subject has a revision PDF inside `C - CAT/Section B/`.
Upload it to the Claude Project so Claude can reference it while teaching.

**Step 5 — Practice MCQs**
After studying any concept, switch to the MCQ Project.
Type a topic name → quiz starts instantly. No setup needed per topic.

## 🧠 Why This Method Works

| ❌ Traditional Approach | ✅ This Method |
|------------------------|---------------|
| Read notes passively | Claude teaches, you respond |
| Attempt MCQs blindly | MCQs only after concept is confirmed |
| Check answers at the end | Instant feedback + deep explanation |
| Don't know your gaps | Claude cross-questions to expose them |
| Forget after one read | Dialogue forces active recall |

> 💡 *"You don't know what you don't know — until someone asks you the right
> follow-up question. Claude does that automatically."*
